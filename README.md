## 目前实现的功能

- [x] `/sh <页码>` 服务器帮助
- [x] `/shm` 管理命令
   - `/shm reload` 重载插件配置文件
   - `/shm version` 插件版本
- [x] `/bread` 给面包
- [x] 玩家进出服的人数提示

## 经过测试可用的版本

- [x] 0.12
- [x] 0.14
